# HTTP Request Methods (HTTP 请求方法) {#httprequestmethodsglossary_http_request_methods_http}

HTTP 定义了一组请求方法, 以表明要对给定资源执行的操作.

参阅: MDN (https://developer.mozilla.org/zh-CN/docs/Web/HTTP/Methods)

## GET {#httprequestmethodsglossary_get}

GET 方法请求一个指定资源的表示形式, 使用 GET 的请求应该只被用于获取数据.

## HEAD {#httprequestmethodsglossary_head}

HEAD 方法请求一个与 GET 请求的响应相同的响应, 但没有响应体.

## POST {#httprequestmethodsglossary_post}

POST 方法用于将实体提交到指定的资源, 通常导致在服务器上的状态变化或副作用.

## PUT {#httprequestmethodsglossary_put}

PUT 方法用请求有效载荷替换目标资源的所有当前表示.

## DELETE {#httprequestmethodsglossary_delete}

DELETE 方法删除指定的资源.

## CONNECT {#httprequestmethodsglossary_connect}

CONNECT 方法建立一个到由目标资源标识的服务器的隧道.

## OPTIONS {#httprequestmethodsglossary_options}

OPTIONS 方法用于描述目标资源的通信选项.

## TRACE {#httprequestmethodsglossary_trace}

TRACE 方法沿着到目标资源的路径执行一个消息环回测试.

## PATCH {#httprequestmethodsglossary_patch}

PATCH 方法用于对资源应用部分修改.
